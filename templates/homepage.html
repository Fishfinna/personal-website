Video Games

A Pygame Videogame

Boopers!

Boopers is a Pong-like video game that I created using Pygame and OOP coding methodologies. It features my own original music and three game modes: Free Play which is PVP without an end, Ranked which is PVP where the first player to 10 points wins, and AI where a player can fight an AI in a ranked battle.

To play, download the .ZIP file below, extract it, and run the .EXE file found inside.
